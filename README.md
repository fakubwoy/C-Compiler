# C-Compiler

## Steps to Run the Project

1. Clone the repository:
   ```bash
   git clone https://github.com/fakubwoy/C-Compiler.git
   ```

2. Change directory into the project:
   ```bash
   cd C-Compiler
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```
